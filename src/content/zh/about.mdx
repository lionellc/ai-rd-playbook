# 关于这个博客

我是企业内部研发场景的实践者，当前重点是把 AI 从“个人效率工具”升级为“团队流程能力”。

## 关注主题

### 1) 多 Agent 协作在研发流程中的真实落地

- 从需求创建、任务拆解、编码、自测、提审到复盘，形成可执行的闭环。
- 明确人机协作边界：哪些环节自动化，哪些环节必须保留人工 Gate。
- 用结果指标验证价值（交付周期、返工率、评审通过率、复盘闭环率）。

### 2) 知识库与 Skill 库的工程化方法

- 知识库分层：业务语境、技术实现、流程规范、历史复盘。
- Skill 标准化：输入约束、执行步骤、输出模板、质量检查。
- 以真实项目反馈持续迭代，把“个人经验”沉淀为“团队资产”。

我更关注“可复用、可度量、可持续”的落地方式，而不是一次性的 Demo 效果。

## 站点规划

- 主线文章：按季度沉淀方法论。
- 扩展文章：围绕工程细节、踩坑复盘、工具链实践。
- 持续迭代：保留实验空间，逐步接入更多工程能力。
